## 🟢 Высокий приоритет — рекомендую купить

### 1. **Trading Economics API** — экономический календарь с consensus

| | |
|---|---|
| **Что даёт** | Автоматический сбор consensus аналитиков для FOMC, CPI, NFP, PMI и ~600 других индикаторов в реальном времени. Историческая база forecasts vs actuals. |
| **Цена** | **$99/мес** (Starter, 10000 req) или **$499/мес** (Professional, 100000 req) |
| **Что решит в системе** | Сейчас MARKET_EXPECTATIONS в `upcoming_events.py` обновляется командой вручную. С API будет авто-обновление каждый день — освобождает аналитика на 4-5 часов работы в месяц. Точные consensus вместо приблизительных. |
| **ROI** | Окупается, если решение принимает 1 эксперт >5 часов/мес × средняя ставка эксперта. |
| **Альтернатива** | Бесплатно через CME FedWatch JSON (только для FOMC) — мы уже даём ссылку. |

### 2. **VPS для постоянной работы** — Selectel / Hetzner

| | |
|---|---|
| **Что даёт** | Свой хостинг вместо Streamlit Cloud: всегда онлайн, кэш не сбрасывается, своё доменное имя, работа из РФ без VPN, контроль ресурсов. |
| **Цена** | **400-1000 ₽/мес** (Selectel/Timeweb), либо **€5/мес** (Hetzner) |

| **ROI** | Время сотрудников × количество заходов в месяц. На команде из 5 человек × 20 заходов = экономия ~1.5 ч/мес. |

### 3. **ICSG Monthly Copper Bulletin** — глобальный баланс меди

| | |
|---|---|
| **Что даёт** | Полный месячный бюллетень: production / refining / consumption по странам, regional balances, forecasts. Сейчас бесплатно — только snapshot. |
| **Цена** | **€2 000 — 3 000/год** (~€200/мес) индивидуальная лицензия |
| **Что решит в системе** | Сейчас фундаментальный баланс (дефицит/профицит) — самая большая «слепая зона» модели. С ICSG: новая фича `icsg_balance_quarter` → улучшение MAPE на 1-3% на горизонтах 3-6 мес. |
| **ROI** | Если ошибка прогноза на 1% при закупке 1000 т меди = 130-150 тыс. ₽ потерь. Точность +1% за квартал — окупает подписку. |

### Итого по «зелёному» уровню

| Бюджет | Эффект |
|---|---|

## 🟡 Средний приоритет — желательно

### 4. **S&P Global Commodity Insights (Platts)** — TC/RC и физика концентратов

| | |
|---|---|
| **Что даёт** | Ежедневный TC/RC benchmark, цены концентратов, физические премии, метрики Chinese smelter утилизации. В нашей записке упоминается как ключевой индикатор spot tightness. |

| | |
|---|---|
| **Что даёт** | Yangshan Premium, китайские TC/RC, bonded zone stocks, спотовые премии, физические транзакции. Сейчас бесплатно — только текущие графики без CSV. |
| **Цена** | **$5 000 — 15 000/год** (Copper Tracker) |
| **Что решит** | Yangshan Premium — самый чувствительный недельный индикатор китайского физического спроса. Добавляется как фича. |
| **ROI** | Высокий ROI для компаний с китайскими контрактами. Меньше — если закупки только в РФ/ЕС. |

### 6. **Anthropic Claude API или OpenAI API** — продвинутый NLP-сентимент

| | |
|---|---|
| **Что даёт** | Замена VADER (общий sentiment) на специализированную LLM, понимающую контекст. Может извлекать структурированные данные из новостей (mine name, country, quantity affected). |
| **Цена** | **$50 — 200/мес** при объёме ~5000 новостей в месяц через Claude Haiku/Sonnet |
| **Что решит** | Сейчас VADER даёт ±0.1 точности тональности. LLM — ±0.02. Также: автоматическое заполнение каталога событий из новостей. |
| **ROI** | Снижает работу аналитика по разметке событий с ~3 ч/мес до 0.5 ч. |

## 💰 Сценарии бюджета

### Сценарий A — «Минимальный production» (~$250/мес = $3 000/год)

- ✅ VPS Selectel ($10/мес)
- ✅ Trading Economics API Starter ($99/мес)
- ✅ ICSG Bulletin (~$170/мес = $2 000/год)

**Что получим:** надёжный хостинг с авто-prewarm, актуальный календарь без ручных правок, фундаментальный баланс рынка как фича модели.

**Ожидаемый эффект:** +5-7% к точности прогноза на 3-6 мес. Освобождает 25 часов аналитика в месяц.

### Сценарий B — «Средний серьёзный» (~$1 500/мес = $18 000/год)

Сценарий A + 
- ✅ S&P Platts Copper TC/RC ($1 000/мес)
- ✅ Anthropic Claude API ($150/мес)
- ✅ SMM China Tracker если работаете с Китаем ($500/мес)

**Что получим:** фундаментал концентратов + китайский физический рынок + продвинутый NLP. Можно автоматизировать разметку каталога событий из новостей.

**Ожидаемый эффект:** +10-15% к точности на длинных горизонтах. Закрывает почти все «слепые зоны» MVP.

### Сценарий C — «Premium institutional» (~$5 000-8 000/мес = $60-100k/год)

Сценарий B +
- ✅ CRU Copper Service ($3 000/мес) или Wood Mackenzie
- ✅ Bloomberg Terminal на 1 рабочее место ($1 800/мес)
- ✅ CEIC China полный доступ ($1 500/мес)

**Кому нужно:** только если компания имеет десятки тысяч тонн закупок в год и собственный департамент аналитики.

## 🎯 Моя рекомендация

**Стартовать с Сценария A** (~$250/мес):

1. Сразу — **VPS** (решает проблему первого запуска и геоблокировки РФ).
2. Через 1-2 недели — **Trading Economics** (автоматизирует календарь).
3. Через месяц после старта — **ICSG Bulletin** (после изучения, как лучше встроить фундаментал).

**Через полгода работы** — оценить эффективность по back-test'у с новыми фичами. Если MAPE улучшилось >3% на 3-мес горизонте — переходить к Сценарию B.

**Сценарий C — только если** будут реальные бизнес-кейсы (например, расширение закупок в Китае на 10× → SMM становится критичным).

## ❓ Часто задаваемые вопросы

### «Можно ли обойтись без всех подписок?»

Да, текущий MVP работает и без них. Сейчас система покрывает ~80% оценочной значимости через бесплатные источники. **Платные подписки добавят ~5-15% точности** — но это не критичный прирост для тестирования концепции.

### «Какая подписка даёт максимум за минимум денег?»

**VPS ($10/мес)** — на порядок улучшает UX (первый запуск 2 сек вместо 60). Все остальные подписки — для качества прогноза, а не для удобства.

### «Что если бюджет 0₽?»

Текущий MVP работает в Streamlit Cloud бесплатно. Можно усилить:
- **UptimeRobot ping** (бесплатно) — против засыпания app.
- **Hugging Face Spaces** (бесплатно) — для обхода блокировки РФ.
- **FRED API ключ** (бесплатная регистрация) — даёт ещё ~6 макрорядов.

### «Подписки — это разовая или ежегодная трата?»

Все указанные подписки — **подписочная модель**, годовой контракт обычно даёт 10-15% скидку. Бюджет планировать как **OPEX**, не CAPEX.

### «Какая подписка решает задачу прогноза цены меди в рублях?»

Никакая отдельная — это **расчётная задача**. Мы получим прогноз в USD, домножаем на текущий курс USD/RUB (бесплатно через ЦБ РФ или Yahoo `USDRUB=X`), плюс прогноз курса (отдельная модель). Это не покупка данных, а **разработка** на ~2 недели.
